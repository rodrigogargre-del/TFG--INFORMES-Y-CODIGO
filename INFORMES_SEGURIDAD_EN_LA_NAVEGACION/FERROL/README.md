# Ejercicios de evaluación – Entorno de Ferrol

Esta carpeta contiene los informes generados automáticamente por el sistema de evaluación de seguridad en la navegación aplicados a ejercicios desarrollados en el entorno del de Ferrol.

Los documentos incluidos corresponden a diferentes ejercicios simulados, evaluados mediante el modelo propuesto en este trabajo. Cada informe analiza el comportamiento del buque propio durante el ejercicio.

Los informes incluyen representaciones y diferentes gráficas de la navegación para la interpretación final. Esto permite identificar el nivel de seguridad y posibles aspectos de mejora.
