# TFG – Evaluador inteligente en el Simulador de Navegación (SIMNAV)

Este repositorio contiene una colección de informes generados durante el desarrollo del Trabajo de Fin de Grado (TFG) centrado en la evaluación automática de la seguridad en la navegación mediante técnicas de análisis de datos y lógica borrosa.

El objetivo de estos documentos es mostrar ejemplos reales de los informes que el sistema evaluador genera tras analizar ejercicios de navegación realizados en simulador.

## Contenido del repositorio

```
INFORMES_SEGURIDAD_EN_LA_NAVEGACIÓN
```

Dentro de esta carpeta se incluyen:

* **5 ejercicios realizados en Ferrol**
* **5 ejercicios realizados en Rota**

Cada informe corresponde a un ejercicio completo analizado por el sistema evaluador y muestra:

* La trayectoria seguida durante la navegación
* La evaluación de seguridad obtenida
* Las evidencias gráficas del análisis
* La interpretación del resultado
* Recomendaciones para la mejora del ejercicio

Los ejercicios incluidos representan diferentes situaciones de navegación con el objetivo de mostrar distintos comportamientos y resultados del sistema de evaluación.

## Contexto del proyecto

Estos informes forman parte del desarrollo de un **evaluador automático de seguridad en la navegación** diseñado para analizar ejercicios realizados en el simulador de navegación de la Escuela Naval Militar.

El sistema analiza la trayectoria del buque y genera informes automáticos que proporcionan retroalimentación al alumno sobre la seguridad de su navegación y los aspectos susceptibles de mejora.
